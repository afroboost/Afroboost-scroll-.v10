<analysis>**original_problem_statement:**
The user wants to transform the existing Afroboost application into a multi-partner SaaS platform. The application owner, Bassi, acts as the Super Admin (, ). The core of the application is a mobile-first, Instagram Reels style vertical video feed.

**PRODUCT REQUIREMENTS (Summary of Missions):**
The development has been driven by a series of missions, each with specific UI/UX and logic goals. The key themes are:
1.  **UI/UX Perfection:** Achieve a minimalist, zero black space, immersive, Reels-style interface with a premium feel (deep black backgrounds, frameless elements, violet glow accents).
2.  **Robust Navigation & Logic:** Implement dynamic UI logic based on user roles and the number of active partners. For example, the Reels-style vertical scroll is only active if more than one partner exists. The Réserver button's behavior changes based on context (scroll on the page vs. navigate).
3.  **Super Admin Controls:** Grant the Super Admin (Bassi) the ability to dynamically control the application, including setting the credit cost for various partner services (Campaigns, AI Chat, Promo Codes).
4.  **Monetization & Partner Lifecycle:**
    *   Implement a credit system where partners purchase credits to use premium services.
    *   Integrate Stripe for selling credit packs directly to partners, with funds going to the Super Admin's account.
    *   Implement a webhook to automatically add credits to a partner's account upon successful payment.
    *   Lock premium services if a partner's credit balance is insufficient.
5.  **PWA & Audits:** Ensure the application is installable as a PWA for a fullscreen mobile experience and conduct audits to ensure core functionalities (prompt isolation, notifications) are working as expected.

**User's preferred language**: français

**what currently exists?**
The application is a full-stack platform with a React frontend and a FastAPI backend. It features a sophisticated UI/UX that has been progressively refined to meet the user's premium minimalist standard.

Key functionalities implemented and stabilized:
-   **Reels-style UI:** A full-screen, vertical video feed that dynamically adapts its behavior (e.g., enabling/disabling scroll) based on the number of active partners. It is now full-width, with no black borders.

-   **PWA:** The application is configured with a manifest and service worker to be installable on mobile devices for a fullscreen experience.
